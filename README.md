1. **Context Acquisition** ‚Üí Query `context-manager` for project state
2. **Task Execution** ‚Üí Perform specialized work with context awareness

```json
  "requesting_agent": "react-pro",
  "request_type": "get_task_briefing",

    "query": "React project context and recent component work"

}
```
